## Installation

Install dependencies:

```bash
pnpm i
```

Install hooks:

```bash
pnpm prepare
```

## Getting Started

Run local development server:

```bash
pnpm dev
```

Lint files:

```bash
pnpm lint
```

Run tests:

```bash
pnpm test
```

Compile files and build bundle:

```bash
pnpm build
```

## Commands

Commit changes using conventional changelog:

```bash
pnpm commit
```
